# 🌬️ AirY

Your Personalized Air Quality Navigator & Guardian Angel.

AirY takes air safety to the next level by combining AI-powered personalization, real-time AQI tracking, and smart routing to protect you from harmful air environments — every breath of the way.

## 🌟 Features

- **✅ Personalized Air Preference Detection**
  Analyzes your sensitivity and comfort levels to air quality through quick onboarding.

- **✅ Real-Time AQI Monitoring**
  Get live AQI readings for your current location or any area of interest.

- **✅ Smart Route Finder**
  Plan a route between two locations that avoids high-pollution zones for a cleaner journey.

- **✅ Instant Alerts in High AQI Zones**
  Receive alerts when you enter high-risk air quality regions.

- **✅ Dynamic Dashboard**
  Visualize pollution levels, safe zones, and personalized recommendations.

- **✅ Mobile-Ready & Web-Based**
  Works smoothly across platforms. Built with React, Vite, and modern map APIs.

- **✅ Seamless UX**
  Smooth animations, interactive maps, and real-time updates for a next-gen, premium experience.

## 🛠️ Installation Guide

You need Node.js and npm installed.

1. **Clone the repository**
   ```bash
   git clone https://github.com/IAK143/AirY.git
   cd AirY
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
